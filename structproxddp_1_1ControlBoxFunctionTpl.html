Detailed Description

template<typename _Scalar>
struct proxddp::ControlBoxFunctionTpl< _Scalar >

A simple function \(f(u) = [u_{\min} - u; u - u_{\max}]\).

This function should be used along proxnlp::NegativeOrthant to create control bound constraints

\[ -u_\min \leq u \leq u_\max. \]
