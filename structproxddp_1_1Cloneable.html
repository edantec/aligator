Detailed Description

template<typename T, typename U = void>
struct proxddp::Cloneable< T, U >

Mixin which makes a class cloneable.

Template Parameters
TThe type to make cloneable, or its parent through which clone() is called.
UThe concrete type to clone, or nothing if T is a top node in the class hierarchy.
Warning
This requires type U to have a copy constructor. The compiler-generated default one will suffice (if it is not deleted).
